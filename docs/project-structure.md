# Project Structure

```text
OpenM5Kit/
```

## `src/`

Shared code compiled as the `OpenM5Kit` Arduino library.

## `examples/`

Deployable sketches for each supported device family.

## `docs/`

Setup, deployment, and project notes for users and contributors.
